# 提交文档说明

本文档用于说明本次课程作业随代码一并提交的文档材料，以及每份材料覆盖的要求。

## 1. 已提交文档

| 文档 | 用途 |
| --- | --- |
| `docs/作业3最终报告.md` | 主报告，说明项目目标、原系统分析、集成设计、XML 数据集成方案、流程图、演示步骤与测试验收结果 |
| `docs/演示与验收说明.md` | 演示和验收脚本，说明启动方式、演示账号、手工演示流程、自动验收命令和验收记录模板 |
| `docs/文档提交清单.md` | 本说明文件，用于快速定位提交材料与作业要求的对应关系 |
| `README.md` | 项目根目录使用说明，提供一键启动、服务端口、演示账号、环境变量和联调链路说明 |

除文档外，仓库还包含：

| 材料 | 路径 | 用途 |
| --- | --- | --- |
| 示例数据与 SQL 脚本 | `data/generated/` | SQL Server、Oracle、MySQL 三类数据库的导入 SQL |
| 源代码 | `college_a/`、`college_b/`、`college_c/`、`integration_server/`、`frontend/` | 三院后端、集成服务器和前端 GUI |
| XML Schema | `integration_server/xsd/` | 统一课程、学生、选课 XML 结构约束 |
| XSLT 文件 | `integration_server/xsl/` | 统一 XML 与目标学院写回/退选结构的转换模板 |
